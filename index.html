Press Space to start or stop. Tap the Tap button in time to set the tempo.

Why this metronome stays in time

Most browser metronomes fire their click from a JavaScript timer, which stutters whenever the page is busy. This one schedules every click ahead of time on the Web Audio hardware clock, so the beat lands exactly where it should — perfect for practising scales, tracking takes, or locking in a groove.

How to use it

Drag the tempo slider or use the − / + buttons to dial in your BPM. Pick a time signature to set how many beats are in each bar; the first beat is accented so you always know where the bar starts. Not sure of the tempo? Hit Tap along with a track and we'll work it out.
